# Models

## Recommended Defaults

- `sonar`: fastest, cheap factual lookups
- `sonar-pro`: default for most research questions
- `sonar-reasoning-pro`: better for multi-step reasoning and code-heavy analysis
- `sonar-deep-research`: exhaustive async research workflow, not included in this MVP wrapper

## Pricing Snapshot

- `sonar`: $1 input / $1 output per 1M tokens
- `sonar-pro`: $3 input / $15 output per 1M tokens
- `sonar-reasoning-pro`: $2 input / $8 output plus reasoning cost
- `sonar-deep-research`: $2 input / $8 output plus reasoning, citation, and search costs

Check Perplexity docs for current pricing before heavy use.
